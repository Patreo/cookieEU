# cookie.eu
jQuery EU cookie law compliance plugin

### Samples
Please see index.html file or https://companhiadosbrinquedos.pt/

### Parameters
See default variable in jQuery plugin

### Usage

    $('body').cookieEU();
    
Or

     $('body').cookieEU({
        message: 'your compliance message',
        buttonText: 'close this'
     });
